# Decimal to Signed Binary

## **Purpose**
The program demonstrates three different methods of representing integers in binary using:
1. **Sign and Magnitude**
2. **One's Complement**
3. **Two's Complement**

Key Method : **Bitwise operation**

Programming Language : **C**
